# BACK END PUCAR

## Feito com:

- squelize
- express
- mariadb
- jwtwebtokens
- nodemailer
- multer

### Dev dependece

- chai
- mocha

# Para inicar como dev

Digite o comando: `yarn` para instalar os pacotes

E apos a conclusão da instalação do pacote digite: `yarn dev` ou digite: `yarn test` para rodar os testes de integração automatizados
